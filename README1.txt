-Les fichiers utiles à la programmation de la bataille navale sont :
	- serveur.py
	- client.py
	- ia.py
	- IA_aux.py
	- aux.py

Les fichiers serveur.py et client.py codent la partie client/serveur du programme, le fichier ia.py code l'algorithme d'intelligence articielle et les fichiers restant sont les fichiers de fonctions auxiliares utiles aux programmes précédents.

- Comment jouer ?
	--> Dans un terminal, executer le script serveur.py
	--> Dans un autre terminal, executez le script client.py
	--> Si vous souhaitez jouer en PvP, executez à nouveau le script client.py dans un 3eme terminal, sinon executez le script client_ia.py dans un troisième script.
	--> Suivez les instructions.
